## Objective

Elevate the product from **robust** to **scale-proof** by covering critical real flows with deterministic tests, ensuring:

- No silent regressions
- Zero real-time dependency
- Zero accidental data mutation
- Stable navigation under failure

**This PI does NOT create features, does NOT alter logic, does NOT touch domain — only validates existing contracts.**

## Inviolable Principles

### Determinism

❌ `Date.now()`
❌ `new Date()` without explicit reference
✅ `FIXED_TIMESTAMP_ISO`
✅ `freezeTime()` at start of EVERY test

### Mutation

🚫 FORBIDDEN: POST | PUT | PATCH | DELETE
🚫 FORBIDDEN: Alter protected tables
✅ Browsing-only, pure read

### Navigation

🚫 No implicit redirect
🚫 No async loop
✅ URL stable ≥ 10s
